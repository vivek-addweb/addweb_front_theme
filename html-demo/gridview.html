Susy Grid

Simple Install

# command line

gem install susy

Install using npm

npm install susy sass-loader --save-dev

Install Susy in Compass:

If you want to use Susy with Compass For new Compass project:

# command line

compass create --using susy

Add Susy to a current project

# command line

compass install susy

Install Susy in Gulp

Install susy with npm:

npm install susy --save-dev

Install Susy in Bower

# command line

bower install susy --save
